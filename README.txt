This is a MapReduce word counting use case implemented in the X10 language. More information regarding X10 can be found in the following website: http://x10-lang.org/

This program is written in X10 using a plugin in Eclipse. Users can refer to documentation regarding running this at the website. In the data folder, sample text files are included. The actual codes are contained in the src folder.

The X10Paper.pdf contains details regarding implementation and performance analysis: X10-Enabled MapReduce,, Han Dong, Shujia Zhou, David P. Grove, Proceedings of the Fourth Conference on Partitioned Global Address Space Programming Model, October 12, 2010, New York, NY
